# 🧠 Smart Notes App

A simple but feature-rich Notes App built using HTML, CSS, and JavaScript.

## ✨ Features

- Create notes
- Edit notes
- Delete notes
- Pin important notes
- Search notes
- Tags (Study, Work, Ideas, General)
- Date & time saved
- Local storage (no database)
- Clean UI

## 📁 Project Structure

smart-notes-app/
│
├── index.html
├── style.css
├── script.js
└── README.md

## 🚀 How to Run
1. Open `index.html`
2. Start writing notes

No installation required.

## 💾 Storage

Notes are saved in:
```
Browser LocalStorage
```

This means:
- Notes stay after refresh
- No internet required
- Works offline

## 🛠️ Tech Stack

- HTML

- LocalStorage
